#  Breast Cancer Detection using Deep Learning

##  Description
Ce projet a pour objectif de détecter le cancer du sein à partir d’images médicales en utilisant des techniques de **Deep Learning**, plus précisément un **réseau de neurones convolutifs (CNN)**.

Le modèle permet de classifier les images en deux classes :
- **Benign (Bénin)**
- **Malignant (Malin)**

Ce travail a été réalisé dans un cadre académique afin d’appliquer les concepts du Deep Learning et de la vision par ordinateur.

---

##  Objectifs du projet
- Charger et prétraiter des images médicales
- Construire un modèle CNN pour la classification binaire
- Entraîner et évaluer le modèle
- Visualiser les performances (accuracy & loss)
- Sauvegarder et réutiliser le modèle entraîné
